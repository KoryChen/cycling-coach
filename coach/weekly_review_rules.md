# 週訓練 Review 規則

這些規則用於評估每週訓練是否符合長期目標、負荷安排是否合理，以及下週應如何調整。
Weekly review 不應只用單週 TSS 判斷好壞，需同時看訓練目的、負荷趨勢、強度分佈、恢復安排與關鍵課完成度。

Review 時應先判斷本週性質（建設週、恢復週、測驗週、比賽週、旅行/維持週、回歸訓練週），再套用下列規則。

---

## 資料品質與脈絡

- 若缺少活動列表、TSS、時間、功率區間、心率區間、CTL / ATL / TSB、睡眠、HRV 或主觀疲勞，需標註哪些結論可信度較低
- 若 FTP 可能過期，週 TSS、IF、功率區間與強度分佈需保守解讀
- 若本週有旅行、生病、工作壓力、天氣異常、器材問題或資料漏同步，需先說明脈絡，不應直接判定訓練品質差
- 若有明確課表或週目標，應以「是否完成目標」為優先評估依據，而不是只看總量

---

## 週目標與訓練方向

- 2026 年主要目標是 FTP 與 CP5 提升，週 review 應關注是否有足夠的有氧基礎、閾值刺激與 VO2max 刺激
- 一般建設週應包含 1–2 次關鍵訓練，其餘以 Z2、有氧耐力或恢復為主
- 恢復週或旅行/維持週可接受 TSS 與時長下降，重點是疲勞下降、節奏維持與避免過度消耗
- 若本週沒有目標賽事，避免為了追求單週 TSS 而犧牲關鍵課品質

## 週訓練量

- 每週預期訓練時數：10–12 小時；低於此範圍需判斷是否為恢復週、旅行週或外在因素造成
- 單週 TSS 增加建議不超過前 4 週平均或前週的 10–15%，除非是刻意安排的訓練營、長騎週或比賽週
- 若週 TSS 明顯低於近期水準，需判斷是合理恢復、訓練中斷，或關鍵課未完成
- 若週 TSS 很高但關鍵課品質下降、後半週疲勞明顯或 TSB 過低，不應評為高品質週
- 高爬升週需與平路週分開判斷；同樣 TSS 下，爬升多可能造成更高肌肉疲勞

## CTL / ATL / TSB 趨勢

- CTL 上升代表訓練負荷累積，但不應為了提高 CTL 犧牲恢復與品質
- 目前 CTL 目標可維持在約 70–85；若已在此區間，優先提升訓練品質與關鍵功率，而不是硬衝更高 CTL
- ATL 快速上升且 TSB 持續低於 -20，代表疲勞累積，需檢查高強度安排是否過密
- TSB < -30：下週應優先恢復、降載或只保留低風險有氧，不建議安排高強度測驗
- TSB > +15：代表恢復充足，若近期訓練連續性良好，可安排高品質關鍵課或測驗
- 若 CTL 下降但本週是恢復週、taper、旅行或生病後回歸，未必是負面訊號

## 強度分佈

- 建設週應避免過多灰區強度；Z3/Tempo 過多可能造成疲勞高但適應不明確
- 若目標是 FTP 提升，每週應確認是否有足夠 Tempo / Sweet Spot / Threshold 刺激，但不應連續多日堆疊
- 若目標是 CP5 / VO2max 提升，每週可安排 1 次高品質 Z5 間歇；疲勞高時應延後或降強度
- Z4/Z5 高強度日後，應安排恢復日或 Z2，避免連續高強度導致品質下降
- 若本週大多為 Z1/Z2，需判斷是刻意恢復、有氧建設，或缺少必要刺激

## 關鍵課完成度

- Weekly review 應先找出本週 1–2 個最重要訓練，而不是平均評估每一騎
- 關鍵課需評估：是否完成目標區間、功率是否穩定、心率反應是否合理、後半段是否衰退
- 若總 TSS 達標但關鍵課失敗，評估應偏向「量達標但品質不足」
- 若總 TSS 未達標但關鍵課完成良好且恢復改善，可評為「低量但高品質」
- 若多次訓練出現功率下降且心率偏高，需標註為疲勞累積、補給不足、配速過快或恢復不足的可能訊號

## 長騎、爬坡與主要區間趨勢

- 若本週包含多段爬坡、長騎或主要 segment，需比較前半週與後半週、前半段與後半段的功率心率趨勢
- 多段爬坡若後半段功率下降超過 5–10%，同時平均心率上升超過 3–5 bpm，需標註效率下降或心率功率脫鉤
- 若長騎後段在相近功率下心率明顯上升，需檢查補給、脫水、溫度、前段配速與前幾日疲勞
- 若同一路線或同類型爬坡連續幾週表現改善，可列為體能或配速能力進步
- 若後半週的相同強度訓練明顯更吃力，代表週內疲勞管理可能不足

## 恢復品質與風險

- 每週至少應有 1–2 天恢復日、休息日或真正低強度日，除非是特殊訓練營安排
- Easy ride 應確認是否真的 easy；若恢復騎 IF、心率或主觀疲勞偏高，需視為恢復品質不足
- 若 HRV 下降、睡眠不足、靜息心率偏高或主觀疲勞高，需降低下週高強度比例
- 連續 2 週以上 TSB 長期偏低、關鍵課品質下降或心率偏高，需建議降載
- 若出現生病、疼痛或異常疲勞，優先建議休息或保守訓練，不用補課追量

## 下週調整

- 若本週完成度高、TSB 可控、關鍵課品質良好，下週可小幅加量或提高關鍵課難度
- 若本週量高但品質下降，下週應維持或降量，先恢復關鍵課品質
- 若本週疲勞明顯或 TSB < -30，下週應降載，保留 Z1/Z2 與恢復，不安排測驗
- 若本週是恢復週且狀態回升，下週可重新進入建設週，但第一個高強度日需保守開始
- 若下週有測驗或重要長騎，本週 review 應明確建議 taper、休息日與最後刺激課安排

## 報告格式

每次 weekly review 請依以下結構輸出：

1. **週摘要**：日期範圍、騎乘次數、總時長、總距離、總爬升、總 TSS
2. **本週性質判斷**：建設週、恢復週、測驗週、比賽週、旅行/維持週或回歸訓練週
3. **資料品質**：缺漏資料、FTP 是否可信、是否有外在脈絡影響
4. **負荷趨勢**：本週 TSS / 時長與前週、近 4 週平均的比較
5. **體能脈絡**：週初與週末 CTL / ATL / TSB 變化
6. **強度分佈**：Z1/Z2/Z3/Z4/Z5 時間比例與是否符合週目標
7. **關鍵課完成度**：本週最重要 1–2 堂訓練的完成品質
8. **長騎 / 爬坡 / 主要區間趨勢**：功率、心率、後半段衰退或進步訊號
9. **亮點**：值得保留的訓練安排或能力進步
10. **風險與待改善**：疲勞、補給、配速、恢復、強度分佈或訓練連續性問題
11. **下週建議**：加量、維持、降載、恢復週、測驗或關鍵課方向
