更新訓練目標：修改 `goals.md` 中的目標、里程碑或個人資料。

## 步驟

### 第一步：讀取現有目標

讀取 `goals.md`，將現有內容摘要顯示給使用者，確認目前設定。

若 `goals.md` 不存在，提示使用者改執行 `/rider-setup` 進行初始設定。

### 第二步：詢問要更新的項目

列出可更新的項目讓使用者選擇：

1. 個人基本資料（體重、每週訓練時數）
2. FTP / CP5 目前值
3. 年度目標（FTP / CP5 目標瓦數）
4. 目標賽事
5. 里程碑（新增、修改、刪除）
6. 備註

等使用者指定要更新哪些項目，再逐一詢問新的值。

### 第三步：確認並寫入

展示更新後的 `goals.md` 完整預覽，確認後覆蓋寫入。

用繁體中文回覆。
